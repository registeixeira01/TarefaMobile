O Ionic é um framework para criar aplicações tanto móveis quanto desktops, sendo aplicações híbridas.
O que seria uma aplicação híbrida?
Existem dois tipos de aplicações; híbrida e nativa, a híbrida é uma aplicação que é construída de maneira alternativa, ou seja ela pode ser rodada tanto em android quanto em IOS, diferente da nativa que só é executada no sistema operacional programado.
O Ionic é um framewor para criar aplicações com alto desempenho utilizando as tecnologias HTML, CSS e JavaScript.
O Ionic tem o foco na experiência, controle e interação do usuário, alem disso, possui um cliente de linha de comando (CLI), onde possibilita criar um projeto de forma rápida e fornece diversos comandos úteis.
